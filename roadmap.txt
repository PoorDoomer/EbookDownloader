-----ANNAAPI----

findBook:

This endpoint returns a list of books based on some search parameters.

    SearchTerm (q): eg title, author, isbn.
    Author (author): Optional Field for filtering results by author if title already known.
    Limit (limit): Max mumber of results to return. Defaults to 20 if not set.
    Langugage (lang): Field for filtering results by language. Defaults to english if none specified. [en, fr, it, de, es].
    Category (cat): Field for filtering results by category. Defaults to all if none specified. [fiction, nonfiction, magazine, comic, musicalscore, other, unknown].`
    FileType (ext): Field for filtering results by file type. Defaults to all if none specified. [epub, pdf, mobi, azw3]
    Sorting (sort): Field for sorting results. Defaults to most relevant if none selected [newest, largest, oldest, smallest, mostRelevant]

downloadBook:
This endpoint returns a list of links to download the selected book.

    Md5 (md5): from the selected book retreived previously from the findBook request

memberDownloadBook:

This endpoint returns :

    a member only link for fast downloads,

    list of non-member links to download the selected book,

    account info regarding quota usage.

    Md5 (md5): from the selected book retreived previously from the findBook request

    MemberKey (mk): key needed to use membership features
